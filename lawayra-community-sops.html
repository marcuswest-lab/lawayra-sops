SOP 01

In-Person Meetups

Community members host local meetups in their cities for past LaWayra guests. Monica coordinates pre/post calls with each host.

Process

  1. Host expresses interest — Community member reaches out via WhatsApp or Circle. Monica
  2. Pre-event call — Monica calls the host to discuss logistics (venue, date, time, RSVP method), share learnings from past meetups, and determine what promo support LW can provide. Key insight: more structure helps, and keep it exclusively for past LW guests. Monica
  3. Create Circle event — Host creates the event in Circle with date, time, location, and description. Juli can assist if needed. Host / Juli
  4. Send Drip email — Draft email announcing the meetup. Get reviewed in WhatsApp group before sending. Broadcast via Drip. If multiple meetups are happening close together, combine into one email to avoid fatigue. Kevin
  5. Social media post — Instagram story at minimum. Feed post if appropriate. Host can share their own content for reposting. Kevin
  6. Host Circle post — Host posts about the event in Circle to drive RSVPs and engagement. Host
  7. Event happens — Host runs the meetup. Take a group photo for recap content. Host
  8. Post-event debrief call — Monica calls the host: what went well, attendance, what to improve for next time. Monica
  9. Post-event recap — Host posts recap in Circle with group photo. Kevin can repost on social media. Host / Kevin
Important: Meetups should be exclusively for past LaWayra guests — not open to the general public.

SOP 02

Virtual Workshops

Community members and facilitators apply to host virtual workshops for the LaWayra community.

Process

  1. Person applies via form — Applicant fills out the Virtual Workshop Form. Responses go to the Answers Spreadsheet. Applicant
  2. WhatsApp notification — Automated notification to WhatsApp circle group when a new submission arrives. Marcus (automation) In progress
  3. Team reaches out — Contact the applicant to discuss topic, format, duration, scheduling, and collect facilitator bio/photo/website for promo materials. Monica
  4. Create Circle event — Include title, date/time in multiple time zones, description, facilitator bio, and Zoom link. Kevin / Juli
  5. Send Drip email — Draft email with workshop details. Review in WhatsApp group first, then broadcast via Drip. Kevin
  6. Host Circle post — Host announces their workshop in Circle to build interest. Host
  7. Social media post — Instagram story and/or feed post promoting the workshop. Kevin
  8. Day-of: tech setup — Open Zoom/Google Meet, ensure recording is running. One LW team member present for support. Take a gallery-view screenshot for recap content. Host + Team
  9. Post-workshop follow-up — Post-event Circle post with screenshot/recap. Track attendance vs. registrations. Monica / Host
Lesson learned: Adam's workshop (Jan 2026) had 18 RSVPs but only 5 attended. Consider adding: a reminder email the day before, a WhatsApp reminder the morning of, and a "save to calendar" link in the Circle event.

SOP 03

Sunday Community Calls

Weekly call for the LaWayra community to reconnect, share integration experiences, and support each other.

Call Details Active

Schedule: Sundays, 1:00 PM – 2:00 PM Colombian time (CDT)

Zoom: us06web.zoom.us/j/82142017248

Moderated by: Marcus — Sam (founder) joins occasionally

First call: March 29, 2026 (16 attendees)

Call Format

  1. Integration reflection — where everyone is at
  2. Group check-in
  3. Breakout rooms for more personal sharing (small groups)
  4. Come back together to discuss what support is needed
  5. Closing

Weekly Process

  1. Create/duplicate Circle event — Recurring event may need manual duplication each week (has had issues not showing in upcoming). Include date, time in 5 time zones, Zoom link. Kevin / Juli
  2. Send Drip email invite — Use the email template (community-call-email.html). Includes date, times (CDT, PDT, EDT, BST, CEST), RSVP link, and Loom: how to RSVP. Send Wed–Thu. Kevin
  3. Circle post reminder — Post in Circle (also triggers email notification to members). Thu–Fri. Marcus / Kevin
  4. WhatsApp group reminders — Send to past cohort WhatsApp groups. Saturday. Juli / Kevin
  5. Instagram story reminder — Post IG story the day before the call. Kevin
  6. Day-of: host call — Open Zoom, moderate, manage breakout rooms, ensure Fathom is recording. Sunday 1 PM COT. Marcus
  7. Post-call — Share Fathom recording link in WhatsApp circle group. Note any follow-ups needed. Marcus
Note: Some members in other time zones (e.g., Dubai — 2 AM) cannot attend. Future consideration: add a second call for other time zones.
SOP 04

Livestreams

Live streaming Finca activities to the Circle community so remote members can participate in real time.

What to Livestream

ActivityScheduleNotes
Yoga Classes7-day retreats: Tue/Wed/Thu at 3 PM COT4-day retreats: Fri/Sat only
Cacao CeremonySunday at 7 PM COTDuring retreat weeks
Do NOT livestream: Rapeh ceremony — not appropriate for streaming.

Process

  1. Check retreat schedule — Confirm which activities are happening this week based on retreat type (7-day vs 4-day). No classes during breaks between retreats. Kevin
  2. Create Circle event — Use "Live Stream" as the location type in Circle. Include activity name, time, and description. Kevin / Juli
  3. Announce — Circle post + Instagram story promoting the upcoming livestream. Kevin
  4. Tech setup — Set up camera at the maloka. Test internet connectivity first. Kevin
  5. Go live — Stream via Circle's built-in livestream feature. Kevin
  6. Post-stream — Save recording if Circle allows. Share replay link in Circle for those who missed it. Kevin
Known issue: Internet reliability at the maloka is inconsistent. The first yoga livestream (4/2/26) was cancelled due to internet being down. Always test connectivity before scheduling. Have a backup plan (e.g., post a pre-recorded video instead).

SOP 05

Six-Week Integration Program

Monica's structured post-retreat integration program. Small cohort-based, with weekly calls, private chat, 1-on-1 sessions, and homework.

Program Components In Development

2-hour weekly group call — 6 weeks of facilitated group sessions

Private Circle chat — Ongoing support between calls for the cohort

1-on-1 session with Monica — One individual session per participant

Homework & action items — Distributed weekly through Circle

Guided meditation videos — Available in Circle (Monica recording)

Pricing: $30 – $50 per program (under discussion)

Content Focus

Conscious habits and daily routines, meditation, yoga, habit stacking, pairing new practices with things you already do, therapy models for post-retreat life, accountability and goal-setting.

Process

  1. Finalize payment mechanism — Evaluating Bold payment links vs. BookingLayer bookable item. Sarah TBD
  2. Set up private Circle space — Investigate Circle's capabilities for private/exclusive events and chat spaces that only show for enrolled members. Need exclusivity — calls and chat should not be visible to non-enrolled members. Check Circle plan limits (~15 spaces max reported). Marcus TBD
  3. Open enrollment — Announce program via Circle, Drip email, and social media. Include: what's included, schedule, pricing, how to sign up. Monica
  4. Collect payment — Via whichever mechanism is set up (Bold/BookingLayer). Sarah
  5. Create private Circle space — Private chat space + event series for this cohort only. Marcus / Juli
  6. Weeks 1–6: Group calls — 2-hour facilitated calls. Topics build on each other week by week. Monica
  7. Weeks 1–6: Homework — Distribute action items and exercises through Circle chat or posts. Monica
  8. Individual sessions — Schedule one 1-on-1 session per participant at their preferred time. Monica
  9. Guided meditation videos — Record and upload to a "Guided Meditations" section in Circle. Monica
  10. Program completion — Graduation/celebration, collect feedback, discuss continued support options. Monica

Shared Process: Sending a Drip Email

Used across meetups, workshops, and Sunday calls. Follow these steps every time.

  1. Draft the email — Write copy with event details, date/time (include multiple time zones), and CTA (RSVP link to Circle event).
  2. Review in WhatsApp group — Share draft for team feedback before sending.
  3. Send test email — Send to yourself first to check formatting and links.
  4. Broadcast via Drip — Send to the appropriate segment/list. Loom walkthrough
  5. Confirm delivery — Check Drip for open/click rates after sending.
