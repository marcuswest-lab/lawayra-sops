SOP 01

In-Person Meetups

Community members host local meetups in their cities for past LaWayra guests. Monica coordinates pre/post calls with each host.

  1. Host expresses interest — Community member reaches out via WhatsApp or Circle. Monica
  2. Pre-event call — Share learnings from past meetups, discuss logistics (venue, time, RSVP), determine promo support. Key insight: more structure helps. Monica
  3. Create Circle event — Host creates the event with date, time, location, description. Host / Juli
  4. Send Drip email — Draft, review in WhatsApp group, then broadcast. Combine multiple nearby meetups into one email. Kevin
  5. Social media post — Instagram story at minimum. Host can share content for reposting. Kevin
  6. Host Circle post — Post in Circle to drive RSVPs. Host
  7. Event happens — Host runs the meetup. Take a group photo for recap. Host
  8. Post-event debrief call — What went well, attendance, improvements for next time. Monica
  9. Post-event recap — Circle post with group photo. Repost on IG. Host / Kevin
Important: Meetups should be exclusively for past LaWayra guests — not open to the general public.

SOP 02

Virtual Workshops

Community members and facilitators apply to host virtual workshops for the LaWayra community.

Workshop Application Form Active

Form: Virtual Workshop Form

Answers: Responses Spreadsheet

  1. Person applies via form — Responses land in Google Sheets. Applicant
  2. WhatsApp notification fires — Automated alert to the WhatsApp circle group. Marcus (automation) In progress
  3. Team reaches out — Discuss topic, format, duration, scheduling. Collect facilitator bio, photo, website. Monica
  4. Create Circle event — Title, date/time in multiple time zones, description, facilitator bio, Zoom link. Kevin / Juli
  5. Send Drip email — Draft, review in WhatsApp, broadcast. Kevin
  6. Host Circle post — Facilitator announces their workshop in Circle. Host
  7. Social media post — IG story and/or feed post. Kevin
  8. Day-of: tech setup — Open Zoom, start recording, one LW team member present for support. Gallery-view screenshot for recap. Host + Team
  9. Post-workshop follow-up — Circle recap post with screenshot. Track attendance vs. RSVPs. Monica / Host
Lesson learned: Adam's workshop (Jan 2026) had 18 RSVPs but only 5 attended. Add: a reminder email the day before, a WhatsApp reminder the morning of, and a "save to calendar" link.

SOP 04

Livestreams

Live streaming Finca activities to Circle so remote members can participate in real time.

What to Livestream Active

Yoga classes: 7-day retreats: Tue/Wed/Thu at 3 PM COT • 4-day retreats: Fri/Sat

Cacao ceremony: Sunday at 7 PM COT (during retreat weeks)

Do NOT livestream: Rapeh ceremony — not appropriate for streaming.

  1. Check retreat schedule — Confirm activities based on retreat type (7-day vs 4-day). No classes during breaks. Kevin
  2. Create Circle event — Use "Live Stream" as location type. Include activity name, time, description. Kevin / Juli
  3. Announce — Circle post + Instagram story. Kevin
  4. Tech setup at maloka — Camera setup. Test internet first. Kevin
  5. Go live — Stream via Circle's built-in livestream feature. Kevin
  6. Post-stream — Save recording, share replay link in Circle. Kevin
Known issue: Internet at the maloka is inconsistent. The first yoga livestream (4/2/26) was cancelled due to internet outage. Always test connectivity. Backup plan: post a pre-recorded video instead.

SOP 05

Six-Week Integration Program

Monica's structured post-retreat integration program. Small cohort-based, with weekly calls, private chat, 1-on-1 sessions, and homework.

Program Overview In Development

2-hour weekly group call — 6 weeks of facilitated group sessions

Private Circle chat — Ongoing cohort support between calls

1-on-1 session with Monica — One individual session per participant

Homework & action items — Distributed weekly through Circle

Guided meditation videos — In Circle (Monica recording)

Pricing: $30 – $50 per program (under discussion)

Content focus: Conscious habits, daily routines, meditation, yoga, habit stacking, therapy models, accountability

Launch Process

  1. Finalize payment mechanism — Evaluating Bold payment links vs. BookingLayer bookable item. Sarah TBD
  2. Set up private Circle space — Private/exclusive events + chat that only enrolled members see. Check Circle plan limits (~15 spaces). Marcus TBD
  3. Open enrollment — Announce via Circle, Drip, and social. Include: schedule, pricing, what's included. Monica
  4. Collect payment — Via Bold/BookingLayer. Sarah
  5. Create private cohort space — Private chat + event series for this cohort only. Marcus / Juli

Weekly Delivery (Weeks 1–6)

  1. Group call — 2-hour facilitated session. Topics build week by week. Monica
  2. Homework — Action items and exercises distributed through Circle. Monica
  3. Individual session — Schedule one 1-on-1 per participant at their preferred time. Monica
  4. Meditation videos — Record guided meditations and upload to Circle. Monica
  5. Program completion — Graduation, collect feedback, discuss continued support. Monica

Shared Process

Sending a Drip Email

Used across meetups, workshops, and Sunday calls. Follow these steps every time.

Draft Email → Review in WhatsApp → Send Test Email → Broadcast via Drip → Check Open/Click Rates
  1. Draft the email — Event details, date/time in multiple time zones, CTA (RSVP link to Circle event).
  2. Review in WhatsApp group — Get team feedback before sending.
  3. Send test email — Send to yourself to check formatting and links.
  4. Broadcast via Drip — Send to the appropriate list. Loom walkthrough
  5. Confirm delivery — Check Drip for open and click rates.
